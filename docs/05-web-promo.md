# 05 · 홍보 웹페이지 (본 저장소)

## 역할

`wowthisischemistry` Next.js 앱은 프로젝트의 **홍보·설명 면**이다.

보여줄 내용의 우선순위:

1. 프로젝트 한 줄 목적 (계산 근거 + 멀티에이전트 검증)
2. “LLM이 직접 추천하지 않는다”는 설계 포인트
3. 파이프라인 개요 (xTB → JSON → Agents → Report)
4. 사례(미세플라스틱 필터)의 가설·한계·추천 요약
5. (가능하면) 데모 링크 / 코드 / 보고서 샘플

## 기술 메모

- Next.js **16** — 학습 데이터와 다를 수 있음. 코드 작성 전 `node_modules/next/dist/docs/` 및 루트 `AGENTS.md`를 확인한다.
- React 19, Tailwind 4
- 진입점: `app/page.tsx`, `app/layout.tsx`

## 웹이 하면 안 되는 것

- 브라우저에서 xTB 로그를 재해석해 점수를 바꾸기
- LLM API만으로 후보 순위를 “데모용으로”라도 확정 제시하기 (근거 JSON 없이)

계산 백엔드가 아직 이 repo에 없다면, UI는 **문서화된 설계·사례 결과**를 정직하게 설명하는 정적/반정적 페이지부터 구현한다.
