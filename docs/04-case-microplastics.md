# 04 · 사례: 풍화 미세플라스틱 제거 필터

## 가설

풍화된 미세플라스틱 표면이 음전하 또는 극성을 띨 수 있다.  
정전기적으로 상호작용할 수 있는 **양전하 필터 활성층**을 쓰면 제거 가능성이 높아질 수 있다.

## 계산 범위 (중요)

- 실제 입자 전체 ❌
- **대리 구조(proxy)** 끼리 비교 ✅ (표면 단순화 + 필터 활성층 단순화)

xTB 결과는 여과 효율의 직접 지표가 아니라 **초기 선별 지표**다.

## 후보 예시

**필터 활성층 (추천 대상):**

- 키토산 코팅
- PEI 코팅
- 4차 암모늄 유도체
- 양이온 전분
- 폴리도파민 유사 코팅

**대조군 (비교 기준, 최종 추천 집합 제외):**

- 중성 셀룰로오스
- 방향족 활성탄 대리 구조

## 입력 메타에 넣을 것

후보 조합명, `.xyz` 경로, 형식, 전하, 후보 역할, 대조군 여부

## 사례 결과 (소감문 기준)

| 구분 | 내용 |
|------|------|
| 1순위 | 키토산 코팅 조합 |
| 차순위 | 4차 암모늄 유도체, 폴리도파민 유사 코팅 |
| 대조군 | 추천이 아닌 비교 기준으로 유지 |

## 보고서에 반드시 쓸 한계

- 입자 크기, 표면 거칠기, 용매, pH, 농도, 교반, 필터 제작 방식 등이 실측을 좌우한다.
- 시스템은 **의사결정 보조**이며 실험 결론을 대체하지 않는다.
